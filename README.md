# AirAware Indoor Air Quality Prediction

Predicting indoor air pollution exposure using outdoor air quality, indoor activities, and environmental factors.

---

## Project Overview

Indoor air pollution is influenced by both outdoor conditions and indoor behaviours such as cooking, ventilation, heating and occupancy.

This project develops machine learning models capable of predicting indoor concentrations of:

- PM2.5
- PM10
- NO₂

using synthetic data generated from literature-backed emission factors and environmental variables.

---

## Objectives

- Generate realistic synthetic IAQ datasets
- Explore pollutant behaviour
- Build predictive models
- Compare linear and ensemble methods
- Develop an interactive dashboard

---

## Dataset

Features include:

- Outdoor PM2.5
- Outdoor PM10
- Outdoor NO₂
- Temperature
- Window status
- Extractor fan
- Cooking events
- Cooking method
- Fuel type
- Smoking exposure
- Room type
- Time of day

Target variables

- Indoor PM2.5
- Indoor PM10
- Indoor NO₂
